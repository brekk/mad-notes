A Char literal is a single character delimited by single quotes. It is a constituent part of [[String|String literals]].

## Construction
```mad
a = 'a'
newline = '\n'

import String from "String"
word = "word"
String.toList(word) // ['w', 'o', 'r', 'd']
```
